AngularJS过滤器使用

Demo1: 字符串转大写

输出：Hello {{'AngularJS'|uppercase}}

Demo1: 数字取两位小数

123.45678 输出：{{123.45678|number:2}}

Demo1: 货币转换

666 输出：{{666|currency}}

Demo1: 日期转换

    myApp.controller("DateController",function($scope){
        $scope.today=new Date();
    });
输出：
{{ today | date:'medium' }}
{{ today | date:'short' }}
{{ today | date:'fullDate' }}
{{ today | date:'longDate' }}
{{ today | date:'mediumDate' }}
{{ today | date:'shortDate' }}
{{ today | date:'mediumTime' }}
{{ today | date:'shortTime' }}
 年份格式化
四位年份：{{ today | date:'yyyy' }}
两位年份：{{ today | date:'yy' }}
一位年份：{{ today | date:'y' }}
年月日时分秒：{{ today | date:'yyyy-MM-dd hh:mm:ss' }}
 月份格式化
英文月份：{{ today | date:'MMMM' }}
英文月份简写：{{ today | date:'MMM' }}
数字月份：{{ today |date:'MM' }}
一年中的第几个月份：{{ today |date:'M' }}
 日期格式化
数字日期：{{ today|date:'dd' }}
一个月中的第几天：{{ today | date:'d' }}
英文星期：{{ today | date:'EEEE' }}
英文星期简写：{{ today | date:'EEE' }}
 小时格式化
24小时制数字小时：{{today|date:'HH'}}
一天中的第几个小时：{{today|date:'H'}}
12小时制数字小时：{{today|date:'hh'}}
上午或下午的第几个小时：{{today|date:'h'}}
 分钟格式化
数字分钟数：{{ today | date:'mm' }}
一个小时中的第几分钟：{{ today | date:'m' }}
 秒数格式化
数字秒数：{{ today | date:'ss' }}
一分钟内的第几秒：{{ today | date:'s' }}
毫秒数：{{ today | date:'.sss' }}
 字符格式化
上下午标识：{{ today | date:'a' }}
四位时区标识：{{ today | date:'Z' }}
下面是一些自定义日期格式的示例：
{{ today | date:'MMMd, y' }}
{{ today | date:'EEEE, d, M' }}
{{ today | date:'hh:mm:ss.sss' }}

Demo1: 数组过滤

['Ari','Lerner','Likes','To','Eat','Pizza']输出包含e字符的元素：{{ ['Ari','Lerner','Likes','To','Eat','Pizza'] | filter:'e' }}

Demo1: 对象过滤过滤

[{ 'name': 'Ari', 'City': 'San Francisco', 'favorite food': 'Pizza' },{ 'name': 'Nate', 'City': 'San Francisco', 'favorite food': 'indian food' }]
输出含有'favorite food': 'Pizza'的对象：
{{ [{ 'name': 'Ari', 'City': 'San Francisco', 'favorite food': 'Pizza' },{ 'name': 'Nate', 'City': 'San Francisco', 'favorite food': 'indian food' }] | filter:{'favorite food': 'Pizza'} }}

Demo1: 自定义函数过滤

    myApp.controller("TestController",function($scope){
        $scope.isCapitalized = function(str) {
            return str[0] == str[0].toUpperCase();
        };
    });
['Ari','likes','to','travel','Yes','AWesome']输出首字母大写的元素：{{ ['Ari','likes','to','travel','Yes','AWesome'] | filter:isCapitalized }}

Demo1: $filter字符串转小写

    var myApp=angular.module("myApp",[]);
    myApp.controller("LowerCaseController",function($scope,$filter){
      $scope.filteredValue=$filter('lowercase')('HELLO ANGULARJS');
    });
输出：{{filteredValue}}

Demo1: json过滤器可以将一个JSON或JavaScript对象转换成字符串

{'name': 'Ari', 'City': 'SanFrancisco'}输出：{{ {'name': 'Ari', 'City': 'SanFrancisco'} | json }}

Demo1: limitTo过滤器会根据传入的参数生成一个新的数组或字符串，新的数组或字符串的长度取决于传入的参数，通过传入参数的正负值来控制从前面还是从后面开始截取。

San Francisco is very cloudy 输出截取字符串的前三个字符：{{ 'San Francisco is very cloudy' | limitTo:3 }}

Demo1: limitTo测试

San Francisco is very cloudy 输出截取字符串的最后六个字符：{{ 'San Francisco is very cloudy' | limitTo:-6 }}

Demo1: limitTo使用返回数组前3个元素

['a','b','c','d','e','f'] 输出：{{ ['a','b','c','d','e','f'] | limitTo:3 }}

Demo1: orderBy过滤器可以用表达式对指定的数组进行排序。

[{ 'name': 'Ari', 'status': 'awake' },{ 'name': 'Q', 'status': 'sleeping' },{ 'name': 'Nate', 'status': 'awake' }]
根据name排序输出：
{{ [{ 'name': 'Ari', 'status': 'awake' },{ 'name': 'Q', 'status': 'sleeping' },{ 'name': 'Nate', 'status': 'awake' }] | orderBy:'name' }}

Demo1: orderBy过滤器第二个参数设置为true反转排序。

[{ 'name': 'Ari', 'status': 'awake' },{ 'name': 'Q', 'status': 'sleeping' },{ 'name': 'Nate', 'status': 'awake' }]
根据name反转排序输出：
{{ [{ 'name': 'Ari', 'status': 'awake' },{ 'name': 'Q', 'status': 'sleeping' },{ 'name': 'Nate', 'status': 'awake' }] | orderBy:'name':true }}

Demo1: 自定义过滤器将字符串首字母大写

    myApp.filter("customizeCapitalize",function(){
       return function(input){
           //首字母大写实现
           if(input){
               return input[0].toUpperCase()+input.slice(1);
           }
       };
   });
hEllo WOrlD首字母大写输出：{{ 'hEllo WOrlD'| lowercase | customizeCapitalize }}
