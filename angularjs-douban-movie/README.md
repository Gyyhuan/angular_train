如何运行：
1、首先安装gulp,具体参考官方文档(http://www.gulpjs.com.cn/)
#npm install --global gulp或者npm install --save-dev gulp
2、运行gulp命令打包程序（gulp默认会执行default指令）
#gulp
3、将构建完后的程序index.html,assets,ngviews部署到服务器上即可访问
